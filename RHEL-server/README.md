# Red Hat Enterprise Linux Home Server Project

RHEL 9 server build for learning system administration, security configuration/system hardening, and enterprise cybersecurity fundamentals.

# System Overview
- Red Hat Linux 9.7 - Virtual Machine
- Hypervisor - QEMU/KVM, virt-manager GUI
- Host Machine - Arch Linux Desktop PC


## Goals
- Proficiency in linux administration on an enterpise operating system
- Learn how to properly configure and deploy services and applications on a server
- Hands-on practice configuring security settings and hardening the system.
- Monitor, log, and automate system activity
